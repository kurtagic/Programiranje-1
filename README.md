## tj Script
batch file za testiranje java datotek s tj.exe. 

### DIRECTORY ORGANISATION DEMO
1. MAIN_DIRECTORY
- directory
  - File.java
  - tests
      - tests
        - test01.in
        - test01.out
        - ...    
- tj.exe
- tjScript.cmd

  use `tjScript.cmd` to run the script, then give the directory path to the `.java` file, you want to test. in the example above it would be `D:\MAIN_DIRECTORY\directory\`
